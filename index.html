Projects & Skills

Golden Goats

Golden Goats is a C++ game where two characters compete to collect coins. When a player grabs a coin, a basic math question appears —if answered correctly, the coin counts; if not, it doesn't. The first player to collect 3 coins wins. The game was developed entirely in C++ using simple graphics.
